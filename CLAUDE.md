## Project Overview

Signal K server plugin for EmpirBus HVAC integration. Communicates with EmpirBus NXT hardware to provide heating, ventilation, and air conditioning control through the Signal K marine data framework.

## Technology Stack

- **Runtime**: Node.js
- **Framework**: Signal K server plugin API
- **Protocol**: EmpirBus NXT (CAN bus / NMEA 2000)
